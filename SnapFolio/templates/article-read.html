Rock, Paper, Scissor in Python

Subash Dhami • May 6, 2024 • 1 min read • 65 views

The Rock, Paper, Scissors game is a fun way to learn basic Python input/output and logic. Below is a simple implementation: 

import random

def get_player_choice():
    choices = ['rock', 'paper', 'scissors']
    player_choice = input("Choose rock, paper, or scissors: ").lower()
    if player_choice not in choices:
        print("Invalid choice!")
        return None
    return player_choice

def get_computer_choice():
    return random.choice(['rock', 'paper', 'scissors'])

def determine_winner(player, computer):
    if player == computer:
        return "It's a tie!"
    elif (player == 'rock' and computer == 'scissors') or \
         (player == 'paper' and computer == 'rock') or \
         (player == 'scissors' and computer == 'paper'):
        return "You win!"
    else:
        return "You lose!"

player = get_player_choice()
if player:
    computer = get_computer_choice()
    print(f"Computer chose: {computer}")
    print(determine_winner(player, computer))

This code uses Python’s random module and basic control flow to make a fun interactive game.
